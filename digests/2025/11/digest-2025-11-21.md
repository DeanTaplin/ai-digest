### Context Engineering for Agents: Filesystem-based memory strategies
**Source**: [LangChain Blog](https://blog.langchain.com/context-engineering-for-agents/)
**Published**: July 2025 (relevant for current deployments)

LangChain's Deep Agents architecture uses filesystems as external memory for long-running agents. Rather than overloading context windows, agents write notes to files (scratchpads), creating a shared workspace for multi-agent collaboration. This approach lets agents persist information beyond their context limits and iterate naturally with full visibility into filesystem state.

**Why it matters**: As agents run longer and handle more complex tasks, context management becomes critical. Filesystem-based scratchpads are a proven pattern now being used in production systems like Claude Code.

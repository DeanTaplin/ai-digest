### Introducing Showboat and Rodney: Tools for Coding Agents to Demo Their Work

**Source**: Simon Willison
**Link**: https://simonwillison.net/2026/Feb/10/showboat-and-rodney/

Simon Willison released two new open-source tools for coding agents: Showboat (a CLI tool for building Markdown demo documents) and Rodney (CLI browser automation). These tools help agents demonstrate their work beyond automated tests by creating visual, step-by-step documentation of what they've built.

**Key Insight**: The challenge isn't just getting agents to write code‚Äîit's having them prove the code works without manual QA. Showboat and Rodney automate the demonstration process.

**Why It Matters**: As coding agents become more productive, we need better tools to verify their output. These lightweight CLI tools integrate into agent workflows to generate trustworthy demos automatically.

### The Two Patterns for Agent Sandbox Connections

**Source**: LangChain
**Link**: https://blog.langchain.com/the-two-patterns-by-which-agents-connect-sandboxes/

LangChain analyzes two architectural patterns for connecting agents to code execution sandboxes: direct connection (agent controls the sandbox) and indirect connection (agent requests sandbox actions). The choice impacts security, flexibility, and user experience.

**Key Insight**: The sandbox pattern you choose fundamentally affects what your agent can do and how users interact with it. There's no one-size-fits-all solution.

**Why It Matters**: Code execution is a critical capability for production agents. Understanding these patterns helps you architect safer, more capable agentic systems.

### LLM Impact on Book Publishing: Quality vs. Quantity

**Source**: Ethan Mollick / NBER
**Link**: https://bsky.app/profile/emollick.bsky.social/post/3mejla7pn3s25

NBER research shows LLMs tripled book releases since 2022, with average quality falling due to AI-generated 'slop'. However, books ranked 100-1,000 per category actually improved, and pre-LLM authors became more productive, creating a net positive for readers who focus on quality content.

**Key Insight**: AI floods markets with low-quality content but simultaneously improves the production of quality content. The middle-market thrives.

**Why It Matters**: This pattern likely applies beyond books to software, documentation, and technical content. Quality signals become more important.

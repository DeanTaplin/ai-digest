### [LocalSearchBench: Agents Struggle with Real-World Multi-Hop Reasoning](https://arxiv.org/abs/2512.07436)
**Summary:** Researchers introduced LocalSearchBench, the first comprehensive benchmark for agentic search in local life services with 1.3M merchant entries and 900 real user queries requiring multi-step reasoning. State-of-the-art DeepSeek-V3.2 achieved only 35.60% correctness, with most models struggling on completeness (60.32%) and faithfulness (30.72%).

**Key Insight:** Even advanced LRMs fail at domain-specific multi-hop reasoning tasks, revealing the gap between general-purpose agents and specialized vertical applications.

**Why It Matters:** Benchmarks using real business scenarios help developers set realistic expectations for agentic search systems and identify where specialized training is needed beyond general-purpose models.

### [MemRec: Collaborative Memory-Augmented Recommender Agents](https://arxiv.org/abs/2601.08816)
**Summary:** MemRec introduces a framework that architecturally decouples reasoning from memory management in recommender systems. A dedicated cost-effective LM_Mem manages a dynamic collaborative memory graph, serving synthesized context to a downstream LLM_Rec. Achieves state-of-the-art performance while supporting diverse deployments including local open-source models.

**Key Insight:** Separating memory management from reasoning creates a new Pareto frontier balancing quality, cost, and privacy‚Äîespecially important for agents handling sensitive user data.

**Why It Matters:** The architectural pattern of specialized memory management modules could apply broadly to agent systems needing efficient, privacy-preserving collaborative signals beyond recommendations.
